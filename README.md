# Walk Courageously With God — Interactive Study App

A Progressive Web App (PWA) for personal Bible study using the *Walk Courageously With God* publication.

## Features
- 📖 Interactive worksheet for all 21 Section 1 chapters (Ch. 1–21)
- 📍 Chronological Bible timeline with world-power bands
- 🔗 WOL deep links — tap any scripture or Watchtower reference to open on WOL
- 💾 Auto-save answers to device storage
- 📤 Export / 📥 Import answers as JSON backup
- 📵 Full offline support via Service Worker
- 📱 Installable on Android & iOS as a home-screen app

## Chapters Covered (Section 1)
Introduction · Enoch · Noah · Sarah · Abraham (×2) · Rebekah · Jacob · Joseph · Shiphrah & Puah · Moses (×2) · Caleb · Joshua · Rahab · Naomi & Ruth · Barak & Deborah · Jael · Gideon · Jephthah · Samson · Samuel
